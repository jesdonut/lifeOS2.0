Changelog

A record of what has been built and when. No patch notes or minor fixes — only things that meaningfully changed what Seratus can do.

v2 — in development

A full rewrite. Modular, clean, built to last. Replaces the v1 monolith entirely.

May 2026
Landing Public landing page with hero, app preview carousel, feature cards, period section, philosophy. Dark and light theme. Scroll reveal animations.
Calendar Month view with 9 built-in categories. Add, edit, and delete events. Drag events between days. Park events for later in the notes sidebar. Week and year views in progress.
Notes Collapsible sidebar available from every screen. Parking lot for unscheduled events. Free-form notes with inline editing.
Period data Full period data model: cycle merging, EWMA-weighted predictions, fertile window, BBT, discharge, symptoms, birth control logging. Works across irregular cycles, long cycles (up to 90 days), and gaps.
Import Browser-based Apple Health XML importer: extracts period flow, spotting, BBT, discharge, ovulation, and contraceptive records. Parsed entirely on your device. Output is a JSON file you download and review before importing.
Settings Profile (name, birthday, timezone, nationalities, locations, currencies, period toggle), calendar category editor (rename, recolor, add, delete), appearance, and data management (export, import, clear).
Core localStorage-based storage with no download-per-save. JSON export and import for full backups. Horizontal swipe navigation between tabs. Day-change detection. Module contract enforced across all features.
Coming next
Period UI Year calendar view, daily logger, stats panel. Integrates the period data model already built.
Calendar Week view (Mon-Sun grid, daily spending per category). Year view (monthly overview, goals, age display).
Finance Income and expense tracking. Country-adaptive fields (Japan: salary, health insurance, pension, resident tax). Daily spending flows in from the calendar automatically.
Bank & savings Multi-country, multi-currency account balances. Fixed deposits, bonds, NISA (Japan), pension contributions.
Currency & FX Lot tracking, exchange rate logging, P&L in base currency.
Import Clue JSON, Natural Cycles CSV, Flo JSON, Seratus v1 JSON.

v1 — the original

The first version of Seratus. A single-file app built for personal use — not for sharing, not for general use. It worked and it proved the concept. v2 is a clean rewrite of everything v1 got right, with the things it got wrong fixed from the start.

v1 features: calendar with events and daily spending, period tracker, finance, bank accounts, savings and deposits (NISA, bonds), currency lot tracking with FX rates, notes sidebar. All in one ~2,500-line JS file.
